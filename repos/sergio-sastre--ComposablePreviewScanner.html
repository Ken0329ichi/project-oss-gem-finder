sergio-sastre/ComposablePreviewScanner

A Compose Multiplatform friendly library to help auto-generate screenshot tests from Composable Previews (e.g. Android, Glance) with any screenshot testing library: JVM-based (i.e. Paparazzi, Roborazzi) as well as Instrumentation-based (i.e. Shot, Dropshots, Android-Testify, etc.).
