# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability within **GitHub OSS Gem Finder**, please do not open a public issue.

Instead, please report it directly via email to:
**ken0329ichi@gmail.com**

Please include:
- Description of the issue
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and work on a prompt resolution. Thank you for helping keep OSS Gem Finder secure!
